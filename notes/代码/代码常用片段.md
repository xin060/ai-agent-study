<!-- 文件流 -->
第1步：我有一份文件，我想打开它
       → with open("文件名", "模式", encoding="utf-8")as f:

第2步：我想从文件里读数据
       → 如果是文本：f.read()
       → 如果是 JSON：json.load(f)      这里需要引入 json 模块
       → 如果是 CSV：csv.reader(f)      这里需要引入 csv 模块

第3步：不用关文件
       → with 会自动关，你不需要 f.close()

<!-- requests请求 -->
import requests  

# get请求
# response = requests.get("http://localhost:8000/history?limit=1&offset=0")等效于下面的
response = requests.get("http://localhost:8000/history",params = {"limit": "1", "offset": "0"})
print(response.status_code)    获取HTTP状态码 
print(response.json())         把服务器返回的JSON数据转换为Python字典


# post请求
response = requests.post("http://localhost:8000/chat",json= {"message":"你好"})
