RAG	                "先检索再回答"的思路	                    像软件工程里的"开闭原则"，是一种设计思路
Agent	            "让 AI 自主规划并执行任务"的思路	         像敏捷开发，是一种方法论
Memory	            "让 AI 记住之前的信息"的思路	            像缓存设计，是一种架构思路
Tool Calling	    "让 AI 调用外部工具"的思路	                像接口设计，是一种设计模式
多 Agent 协同	     "多个 AI 分工合作完成任务"的思路	          像微服务架构
Prompt Engineering	"怎么写提示词让 AI 表现更好"	            像产品需求文档
Workflow	        "把任务拆成固定流程执行"	                像流水线设计
